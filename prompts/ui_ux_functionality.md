Breaking points functionailty:

mobile (max-width:768px):
Lessons Grid - 1 lesson in each row. When opening one lesson, only it is open.
topics-nav - horizontal scroll
Upper menu - adjust height topics nav changees height position 

tablet (max-width:1024px):
Lessons Grid - 2 lesson in each row. When opening one lesson in a row, the other also opens and has an identical click affect.
topics-nav - horizontal scroll
Upper menu - adjust height topics nav changees height position 

Desktop (greater than 1024px):
Lessons Grid - 2 lesson in each row. When opening one lesson in a row, the other also opens and has an identical click affect.
topics-nav - no horizontal scrolling
Upper menu - no need to adjust height


dropdown: click-only functionailty to have the same behavior in all screen sized. click to open, click to close, no hover.
